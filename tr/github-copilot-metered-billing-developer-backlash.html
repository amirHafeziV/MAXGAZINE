GitHub Copilot'un İlk Sayaçlı Fatura Döngüsü Güçlü Kullanıcılarda 10-50x Artışla Kapandı

30 Haziran, GitHub Copilot'un 1 Haziran'da sabit ücretli aboneliklerden kullanım tabanlı AI Credits sistemine geçişinin ardından ilk tam aylık fatura döngüsünün kapandığı tarihtir. Ajansal özellikleri kullanan geliştiriciler, aylık faturalarının 29 dolardan 750 dolara ve daha yükseğe çıktığını bildiriyor.

1 Haziran 2026'da GitHub, Copilot'u sabit ücretli abonelik modelinden — bireyler için aylık 19 dolar, işletmeler için 39 dolar — kullanım tabanlı GitHub AI Credits sistemine geçirdi. Kod tamamlama ve Next Edit önerileri tüm ücretli planlarda ücretsiz kaldı; ancak ajansal oturumlar, premium sınır model erişimi, çok adımlı otonom görevler ve otomatik kod incelemesi artık sayaçlı hale geldi.

Fatura Gerçekliği

GitHub CPO Mario Rodriguez, tek bir uzun ajansal oturumun şirkete standart bir aylık sohbet sorgusunun maliyeti kadar yük bindirebildiğini önceden kabul etmişti. Platformlarda paylaşılan ekran görüntüleri, 29 dolardan 750 dolara ve 50 dolardan 3.000 dolara fırlayan faturaları gösteriyor. Tek bir kod tabanı yeniden yapılandırma oturumu Claude Opus 4.8 veya GPT-5.6 kullanılarak 30-40 dolarlık AI Kredisi tüketebiliyor.

Sabit Ücretli AI Çağının Sonu

GitHub'ın değişimi, 2022 sonunda başlayan sabit ücretli AI çağının ürün katmanında sona erdiğinin en net sinyalidir. Cursor Haziran 2025'te, Windsurf ise Mart 2026'da benzer adımları atmıştı. En rahatsız edici etki, güvenlik ağının ortadan kalkmasıdır: Kredi bitince premium özellikler tamamen durur. Bu durum Claude Code gibi rakip araçlara göçü hızlandırdı.
